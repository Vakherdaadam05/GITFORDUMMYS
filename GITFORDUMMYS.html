A GIT

A Git egy ingyenes és nyílt forráskódú elosztott verzióvezérlő rendszer, amelyet a kis és a nagyon nagy projektek gyors és hatékony kezelésére terveztek. A Git könnyen megtanulható, és apró lábnyoma van, villámgyors teljesítménnyel. Az olyan SCM eszközöket felülmúlja, mint a Subversion, a CVS, a Perforce és a ClearCase olyan funkciókkal, mint az olcsó helyi elágazás, a kényelmes állomáshelyek és a több munkafolyamat.
